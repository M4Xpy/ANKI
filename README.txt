* This project simplifies the work of self-creation of ANKI cards.

* Finds various translations, pages with google translation, pages with google photos.
 In one click, creates a new card for the given word.

* To start, just run the file run_program.py .

* This is my first project, help and constructive criticism are welcome.

EN UA

* Цей проект спрощує роботу самостійного створення карток ANKI.

* Знаходить різні переклади, сторінки з перекладом Google, сторінки з фотографіями Google.
  В один клік створює нову картку для заданого слова.

* Для початку просто запустіть файл run_program.py.

* Це мій перший проект, допомога та конструктивна критика вітаються.
